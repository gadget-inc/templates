# Getting started

A list of steps to follow to get the template running:

1. [Pull your Gadget project down to your local machine](https://docs.gadget.dev/guides/development-tools/cli#getting-started) using `ggt dev`.

2. **Install dependencies for the mini:**

   ```bash
   cd minis/shop-mini-globe-demo
   npm install
   ```

3. **Link the mini to a Shopify org** (only needed the first time):

   ```bash
   npx shop-minis setup
   ```

   Sign into Shopify and associate the mini with an org. This writes a `.env` file in `minis/shop-mini-globe-demo/` containing your Shop Minis Admin API key.

4. **Configure environment variables** in your Gadget app (via the dashboard or `ggt var set <NAME> <VALUE>`):

   | Variable | Purpose |
   | --- | --- |
   | `SHOP_MINIS_ADMIN_API_KEY` | Copied from `minis/shop-mini-globe-demo/.env` after running `shop-minis setup`. Used by `POST /shop-mini/auth` to verify Shop Mini tokens. |
   | `SHIP24_API_KEY` | [Ship24](https://www.ship24.com/) API key. Used by `POST /shop-mini/tracking` to look up shipment events. |
   | `GEOCODING_API_KEY` | [geocode.maps.co](https://geocode.maps.co/) API key. Used to convert event location strings into lat/lng. |
   | `GADGET_ENVIRONMENT_JWT_SIGNING_KEY` | Auto-generated by Gadget. Used to sign the JWT returned by the auth route. |

5. **Update the Gadget domain in the mini.** Replace `<your-gadget-app>` with your Gadget app name in:

   - `minis/shop-mini-globe-demo/src/config.ts` (`GADGET_API_BASE`)
   - `minis/shop-mini-globe-demo/src/manifest.json` (under `trusted_domains`)

   Leave `cdn.jsdelivr.net` in `trusted_domains`. It's required by the globe component for loading texture assets.

6. **Run the mini:**

   ```bash
   cd minis/shop-mini-globe-demo
   npx shop-minis dev
   ```

   With the dev server running, press:
   - `i`: open in iOS Simulator
   - `a`: open in Android Emulator
   - `q`: show a QR code for testing on a physical device

7. **Try it out.** Enter a real tracking number to fetch live Ship24 data, or type `TEST` as the tracking number to draw a randomized demo path without burning Ship24/geocoding API quota.
